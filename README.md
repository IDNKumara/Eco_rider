Application Name: EcoRide

--------------------------------------
1. OVERVIEW
--------------------------------------
This application is built using Java 21 and JavaFX 21.  
To run the application successfully, you must have the required Java and JavaFX versions installed and correctly configured.

--------------------------------------
2. REQUIRED SOFTWARE
--------------------------------------

1. **Java Version (JDK):**
   - java-21-openjdk-21.0.9.0.10-1.win.jdk.x86_64
   - Download and install the correct JDK for your Windows x64 system.

2. **JavaFX Version (SDK):**
   - openjfx-21.0.9_windows-x64_bin-sdk
   - Extract the JavaFX SDK to a known directory on your computer.
